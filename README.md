# Analysis-Design

# CS 300 Portfolio Reflection

## Project Overview
For this course, I completed projects focused on data structures and algorithms. The main goal was to create a program that could load, organize, search, and print Computer Science course information in alphanumeric order.
Project One focused on alazying the run-time and memory usage of different data structures, including vectors, hash tables, and binary search trees. Project Two focused on building a working program that used a binary search tree to store and sort course information.


## What Was the Problem I Was Solving?
The problem I was solving was how to efficiently store and retrieve course information. The program needed to read course data from a file, allow a user to search for a specific course, and print a sorted list of all courses in the Computer Science program.

## How Did I Approach the Problem?
I approached the problem by comparing different data structures and thinking about how each one would affect performance. Data structures are important because they determine how efficiently a program can store, search and organize information. I chose to use a binary search tree because it allows course information to be inserted and searched efficiently. It also makes it possible to print the courses in alphanumeric order using an in-order traversal.

## How Did I Overcome Roadblocks?
Some of the main roadblocks I faced involved building the binary search tree correctly, loading the CSV file, and making sure the courses printed in the correct order. I worked through these problems by testing each part of the program separately and fixing errors one step at at time. Debugging helped me better understand how the tree structure works and how important it is to check that data is being loaded and stored correctly.

## How Has This Expanded My Approach to Software Design?
This project helped me understand that software design is not just about writing code that works. It is also about choosing the right structure for the problem. I learned to think more about efficiency, organization, and how the program might need to grow or change in the future.

## How Has This Improved My Programming Practices?
My work in this course helped me improve the way I write programs that are maintainable, readable, and adaptable. I now try to use clear variable names, separate my code into organized functions, and write code that is easier to test and update. Overall, this project helped me become more confident with data structures and showed me how important they are in real-world software development.
